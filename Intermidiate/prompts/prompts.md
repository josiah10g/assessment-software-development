## Prompt 1
**AI Tool Used:** ChatGPT

 ## Prompt:
"Generate JavaScript code for a frontend Expense Tracker that captures transactions from a form, stores them in an array, displays a list of transactions, and calculates total income, total expenses, and balance."

## Intent:
To implement real functionality in the Expense Tracker without a backend or any  Javascript frameworks.

## Outcome:
ChatGPT generated base JS code for handling form submissions and updating the UI. I modified variable names, added IDs to match my HTML, and structured the code to fit my project folder.
